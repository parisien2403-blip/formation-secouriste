# Bilan Secouriste — 4° RIISC

Application mobile (PWA) — releveur talkie, bilan victime.

## Fichiers du dossier

| Fichier | Rôle |
|---------|------|
| `index.html` | Application |
| `manifest.json` | Installation PWA |
| `sw.js` | Mode hors-ligne |
| `logo.jpg` | Icône sur le téléphone (image 1780441905134) |
| `phonix.png` | Logo page d'accueil |
| `1780441905134.png` | Source pour régénérer `logo.jpg` |
| `icon-192.png`, `icon-512.png` | Icônes (optionnel) |
| `regen-icons.ps1` | Regénère `logo.jpg` après changement d'image |

## Lancer en local

`LANCER-SECOURISME.bat` à la racine du projet, ou :

http://localhost:8080/formation-secouriste/

## GitHub

Pour installer sur téléphone comme une vraie app : voir aussi le dossier `bilan-secouriste-GITHUB` (dépôt séparé, fichiers à la racine).
